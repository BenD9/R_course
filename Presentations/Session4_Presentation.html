Functions in R

  • name_fn <- function(attributes){
    function_body
    return(object) ## The object will be returned
    }

  • save function in a file: fn_file.R

  • load the functions using: source(“fn_file.R”) ## Make sure the file is in the current path

Functions to save data structure

  • save(data, file = “file.RData”): writes an external representation of R data

  • saveRDS(data, file = file.rds): write a single R object to a file

  • save.image(file = “file.RData”): saves the current environment

  • savehistory(file = “.Rhistory”): saves the commands history

To Remove an object from the environment

  • remove(object)

  • remove(list = ls()): Removes all objects

  • gc(): Clear RAM after removal of object
