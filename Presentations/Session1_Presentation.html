Session 1: Introduction to Rstudio

Vectors

  • Vector is a collection of scalars (E.g. 4 is a scalar; c(4, 3, 5) is a vector)

  • Scalar are 0-dimensional. Vectors are 1-D.

  • Four major types of vectors

  1. Numerical: c(4, 3, 5)
  2. Character: c(“A”, “B”, “C”) (Character scalar is called a “string”)
  3. Factor (Grouping): factor(”Red”, “Red”, “Blue”, “Red”, “Blue”)
  4. Logical: TRUE, FALSE, TRUE, TRUE, FALSE

Function

  • Used to perform a specific operation.

  • Input (Arguments) -> function -> output

  • output <- name(arguments)

  • mode, length, log(), sqrt()

Subsetting

  • Select or extract values

  • Use square brackets: [ ]

  • Subsetting can be positional / condition (by name)
